Dans ce TP j’ai utilisé les APIs RESTful.
J’ai commencé par comprendre le fonctionnement des endpoints, ensuite j’ai développé un script pour envoyer des requêtes HTTP et récupérer les données météo, puis afficher les informations pour une ville, comme la température et l’humidité. 
j’ai amélioré la requête pour obtenir des données en unités métriques et en langue française. 
Ensuite j’ai refait le même travail avec axios à la place de fetch afin de comparer les deux méthodes.
Enfin j’ai testé d’autres APIs REST comme Open Library et la NASA API et RandomUser API
